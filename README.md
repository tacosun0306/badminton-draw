# 羽球抽籤系統

這是一個專為16人羽球團體設計的抽籤系統網頁應用程式。

## 功能特色

- **16位球員管理**：可手動輸入球員姓名或使用隨機產生功能
- **第一輪順序分組**：按照球員順序自動分配（1-4號球場A，5-8號球場B）
- **第二輪亂數抽籤**：從等候區隨機選擇8人上場
- **智能調度系統**：優先安排打較少場次的球員，確保公平性
- **即時狀態顯示**：顯示當前場上球員、等候區球員及各自已打場次
- **完整歷史記錄**：保存每一輪的比賽配對記錄

## 使用方法

1. 在瀏覽器中開啟 `index.html`
2. 輸入16位球員的姓名（或點擊「隨機產生名字」按鈕）
3. 點擊「開始第一輪（按順序）」開始比賽
4. 第一輪結束後，點擊「開始下一輪（亂數抽籤）」進行後續輪次
5. 可隨時點擊「重新開始」清除所有記錄

## 比賽規則

- 每場比賽4人，同時進行2場（共8人在場）
- 第一輪按照輸入順序分組
- 第二輪起採用亂數抽籤，先下場的球員先抽籤
- 系統會盡量避免重複，優先安排打較少場次的球員

## 技術說明

- 純HTML + CSS + JavaScript實作
- 無需伺服器，直接在瀏覽器中運行
- 響應式設計，支援各種螢幕尺寸
